# Запуск проєкту

[← README](../README.md)

## Локальний сервер

Перейдіть у папку проєкту та запустіть вбудований PHP сервер:

```bash
cd php-labs
php -S localhost:8000
```

Відкрийте в браузері: <http://localhost:8000>

## Зупинка сервера

Натисніть `Ctrl + C` в терміналі.

## Запуск на іншому порті

Якщо порт 8000 зайнятий:

```bash
php -S localhost:8080
```

## Запуск конкретного файлу

Без сервера (тільки в терміналі):

```bash
php filename.php
```
